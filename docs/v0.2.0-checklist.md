# v0.2.0 Implementation Checklist

This milestone converts the original MSP reporting requirements into production work items. Version `0.1.x` established a working collector-to-Influx-to-Grafana pipeline. Version `0.2.0` focuses on completeness, configurability, and dashboard coverage.

## Collector Completeness

- [x] Add API-version-aware endpoint discovery and endpoint fallback support.
- [x] Make REST pagination configurable globally and per endpoint.
- [x] Collect all jobs without relying on direct hypervisor or storage APIs.
- [x] Collect Backup, Backup Copy, Replication, Tape, SureBackup, NAS, Agent, and Other job categories.
- [x] Collect sessions and task sessions with full failure context.
- [x] Collect restore point count, latest restore point age, repository, and backup copy status.
- [x] Collect repositories, scale-out repositories, object storage, capacity, free space, used space, and immutability.
- [x] Collect infrastructure: proxies, gateway servers, mount servers, WAN accelerators.
- [x] Collect tape jobs, libraries, pools, media, capacity, and errors where Veeam REST exposes them.
- [x] Collect Veeam license information.
- [x] Collect Veeam version information.
- [x] Resolve VM inventory endpoint differences across supported Veeam REST API versions.
- [x] Emit explicit endpoint availability metrics for unsupported, forbidden, or missing Veeam endpoints.

## Protection Analysis

- [x] Report protected VMs.
- [x] Report unprotected VMs.
- [x] Report VMs with no backup.
- [x] Report stale backup status.
- [x] Report stale replication status.
- [x] Classify VM protection as Backup, Backup Copy, Replication, Both Backup and Replication, or Unprotected.
