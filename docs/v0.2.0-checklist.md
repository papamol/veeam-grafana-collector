# v0.2.0 Implementation Checklist

This milestone converts the original MSP reporting requirements into production work items. Version `0.1.x` established a working collector-to-Influx-to-Grafana pipeline. Version `0.2.0` focuses on completeness, configurability, and dashboard coverage.

## Collector Completeness

- [ ] Add API-version-aware endpoint discovery and endpoint fallback support.
- [ ] Make REST pagination configurable globally and per endpoint.
- [ ] Collect all jobs without relying on direct hypervisor or storage APIs.
- [ ] Collect Backup, Backup Copy, Replication, Tape, SureBackup, NAS, Agent, and Other job categories.
- [ ] Collect sessions and task sessions with full failure context.
- [ ] Collect restore point count, latest restore point age, repository, and backup copy status.
- [ ] Collect repositories, scale-out repositories, object storage, capacity, free space, used space, and immutability.
- [ ] Collect infrastructure: proxies, gateway servers, mount servers, WAN accelerators.
- [ ] Collect tape jobs, libraries, pools, media, capacity, and errors where Veeam REST exposes them.
- [ ] Collect Veeam license information.
- [ ] Collect Veeam version information.
- [ ] Resolve VM inventory endpoint differences across supported Veeam REST API versions.
- [x] Emit explicit endpoint availability metrics for unsupported, forbidden, or missing Veeam endpoints.

## Protection Analysis

- [ ] Report protected VMs.
- [ ] Report unprotected VMs.
- [ ] Report VMs with no backup.
- [ ] Report stale backup status.
- [ ] Report stale replication status.
- [ ] Classify VM protection as Backup, Backup Copy, Replication, Both Backup and Replication, or Unprotected.
